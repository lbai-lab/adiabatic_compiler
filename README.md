# Compling Gate-based Quantum Circuit to Adibatic-based Quantum Circuit

## Installation

Tested with Python 3.10
```
pip install -r requirements.txt
pip install -e .
```

## Running tests
Run this line in the terminal
```
python -m unittest discover test -v -f
```
